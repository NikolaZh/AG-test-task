## Комментарии:

При работе использовались: ChatGPT 5.4 Thinking, встроенный в VsCode автокомпит.

Примеры промптов для GPT:

- создай конфинг роутинга для react-router, для '/' и для '/products', путь '/products' должен быть защищен авторизацией
- создай кастомные хуки useSessionStorageHook и useLocalStorageHook. используй ts
- собери компонент формы для авторизации (поля: логин, пароль, чекбокс "запомнить меня"), используй библиотеки: shadcn для react, react-hook-form, zod для валидации
- создай форму добавления товара (поля: наименование, цена, вендор, артикул)
- создай кастомный хук useDebounce
